# ChessKidoo — Coach Report Builder

A beautiful, light-weight, single-page utility for ChessKidoo coaches to build and export professional session progress reports for students.

## Features

- **5-Star Progress Rating**: Rate students across key areas (Opening, Tactics, Calculation, Endgame, Board Awareness, Focus).
- **Interactive Live Preview**: See the report card update in real-time before downloading.
- **Custom Logo Upload**: Click the logo area to upload a custom academy logo.
- **Persistent Storage**:
  - The uploaded academy logo is saved in `localStorage` so it persists across page reloads.
  - Selected coach name is remembered, saving time on subsequent reports.
- **Clearing & Resetting**: Easily clear student details and star ratings to start a new report card.
- **Date Validation**: Visual warnings if the next class date is set before the current class date.
- **Responsive Layout**: Works on desktops, tablets, and mobile devices (includes auto-scroll to preview on smaller screens).
- **Print to PDF**: Seamless layout formatting optimized for print/Save as PDF, automatically hiding form controls and buttons.

## How to Run

1. Clone or download this repository.
2. Double-click the [index.html](index.html) file to open it in any modern web browser.
3. (Optional) Host it directly on **GitHub Pages** for quick online access.
